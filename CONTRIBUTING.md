# Contributing to Physio Decision Models

Thank you for your interest in this project.

## System Status

This repository is a **frozen reference model** (v1.0.0). The normative documents
(CLINICAL_DECISION_SYSTEM.md, DECISION_ENFORCEMENT_RULES.md, AUTHORITY_SOURCES.md)
are locked and cannot be modified without a formal re-versioning process.

## What You Can Contribute

### Accepted contributions
- Bug fixes in documentation (typos, broken links, formatting)
- Translations of non-normative documents
- Additional models in `models/` that extend (but do not modify) the existing framework
- Additional threshold definitions in `thresholds/`
- Improvements to machine-readable files (llms.txt, llms-ctx.txt, JSON-LD)
- Discussion and academic critique via GitHub Issues or Discussions

### Not accepted
- Modifications to frozen normative documents
- Treatment protocols, exercise lists, or procedural content
- Marketing or promotional content
- Patient-facing materials
- Changes that expand the system scope beyond its defined boundaries

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request with a clear description of what and why

## Code of Conduct

This is a clinical decision framework. Contributions must be:
- Evidence-based and academically rigorous
- Non-promotional
- Respectful of the system's epistemic boundaries

## Questions?

Open a GitHub Issue or start a Discussion.
