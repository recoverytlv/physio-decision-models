## كيف يعمل النظام

الدورة السريرية: **الاستماع ← التقييم ← تحديد الأهداف ← العلاج ← إعادة التقييم ← القرار**

كل حالة يجب أن تنتهي بأحد ستة مخرجات:

| المخرج | المعنى |
|---|---|
| **DECLINE** | رفض — خارج نطاق العلاج الطبيعي |
| **DEFER** | تأجيل — بيانات غير كافية لاتخاذ قرار |
| **REFER** | تحويل — علامات حمراء تتطلب طبيب مختص |
| **TRIAL** | قبول مشروط لـ 3-5 جلسات |
| **CONTINUE** | استمرار — فقط مع تحسن موضوعي موثّق |
| **DISCHARGE** | إنهاء — تم تحقيق الأهداف أو توقف التقدم |

استمرار العلاج يتطلب **أدلة موضوعية**: تحسن بنسبة 10% على الأقل في مدى الحركة، تقدم وظيفي موثّق، أو تقليل في استراتيجيات التعويض.

## المبادئ

1. **كل تدخل هو اختبار فرضية** — تدخل مُتحكَّم به لاختبار فرضية سريرية، وليس تكرار تمارين بدون هدف.

2. **عدم التحسن هو معلومة تشخيصية** — إذا لم يساعد العلاج، فهذه بيانات تُضيّق مساحة الفرضيات. ليس فشلاً.

3. **عدم اليقين يفضّل عدم الفعل** — عند الشك، النظام يختار DECLINE أو DEFER.

4. **القرارات الصحيحة أهم من عدد الجلسات** — النظام يعطي أولوية للقرارات الدقيقة ومعايير الخروج الواضحة.

## علاج طبيعي خاص في تل أبيب

Recovery TLV هي **عيادة علاج طبيعي خاصة** في تل أبيب:

- **جلسات فردية** مع أخصائي علاج طبيعي مخصص
- **تقييم شامل** مع نظام قرارات سريرية رسمي
- **معايير موضوعية** — استمرار العلاج يعتمد على نتائج قابلة للقياس
- **وصول مباشر** بدون حاجة لتحويل طبي للعلاج الخاص في إسرائيل
- **خدمة بالعربية، الإنجليزية، الإسبانية والعبرية**

## الحالات التي نعالجها

- آلام الظهر، انزلاق غضروفي، عرق النسا، تضيق العمود الفقري
- آلام الرقبة، اعتلال الجذور العنقية، إصابة الرقبة السوطية
- آلام الكتف (الكفة المدورة، الكتف المتجمدة، عدم استقرار الكتف)
- آلام الركبة (رضفي فخذي، غضروف هلالي، رباط صليبي، استبدال الركبة)
- آلام الورك (انحشار، اعتلال أوتار الألوية، استبدال الورك)
- التهاب اللفافة الأخمصية، اعتلال وتر أخيل
- مرفق التنس، متلازمة النفق الرسغي
- التواء الكاحل، آلام الصدر، اضطراب المفصل الفكي الصدغي
- إعادة التأهيل بعد الجراحة
- الإصابات الرياضية، متلازمة ما بعد الارتجاج
