## Comment ça fonctionne

Le cycle clinique : **Écouter → Évaluer → Fixer les objectifs → Traiter → Réévaluer → Décider**

Chaque cas doit se terminer par l'une des six sorties :

| Sortie | Signification |
|---|---|
| **DECLINE** | Refus — hors du champ de la physiothérapie |
| **DEFER** | Report — données insuffisantes pour décider |
| **REFER** | Orientation — drapeaux rouges nécessitant un spécialiste |
| **TRIAL** | Acceptation conditionnelle pour 3–5 séances |
| **CONTINUE** | Poursuite — uniquement avec amélioration objective documentée |
| **DISCHARGE** | Fin du traitement — objectifs atteints ou plateau de progression |

La poursuite du traitement nécessite des **preuves objectives** : amélioration d'au moins 10% de l'amplitude de mouvement, progrès fonctionnel documenté, ou réduction des stratégies de compensation. Les rapports subjectifs seuls sont insuffisants.

## Principes

1. **Chaque intervention est un test d'hypothèse** — une perturbation contrôlée pour tester une hypothèse clinique, pas une répétition routinière.

2. **L'absence d'amélioration est une information diagnostique** — si le traitement n'aide pas, c'est une donnée qui restreint l'espace des hypothèses. Ce n'est pas un échec.

3. **L'incertitude favorise la non-action** — en cas de doute, le système privilégie DECLINE ou DEFER. L'optimisme n'est pas un critère clinique.

4. **Les bonnes décisions priment sur le nombre de séances** — le système priorise les décisions précises et les critères de sortie explicites.

## Pathologies traitées

- Douleur lombaire, hernie discale, sciatique, sténose spinale
- Douleur cervicale, radiculopathie cervicale, coup du lapin
- Douleur d'épaule (coiffe des rotateurs, épaule gelée, instabilité)
- Douleur du genou (fémoro-patellaire, ménisque, ligament croisé antérieur, prothèse totale)
- Douleur de hanche (conflit, tendinopathie glutéale, prothèse totale)
- Fasciite plantaire, tendinopathie d'Achille
- Épicondylalgie (tennis elbow), syndrome du canal carpien
- Entorse de cheville, douleur thoracique, dysfonction de l'ATM
- Rééducation post-chirurgicale
- Blessures sportives, syndrome post-commotionnel
