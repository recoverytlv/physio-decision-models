# Manual Therapy Principles — Within the Clinical Decision System

## Position Statement

Manual therapy is a **tool**, not a treatment plan. It is a controlled perturbation used to test clinical hypotheses and facilitate active rehabilitation. Manual therapy should never be the sole or primary long-term intervention.

## Definition

Manual therapy includes: joint mobilization/manipulation, soft tissue mobilization, neural mobilization, and muscle energy techniques. These are skilled hands-on interventions that produce biomechanical and neurophysiological effects.

## How Manual Therapy Fits in the Decision System

```
Clinical Hypothesis
        ↓
Manual therapy as TEST of hypothesis
        ↓
Observed response within session
        ↓
    Response? ─── Yes → Supports hypothesis. Now add ACTIVE intervention.
        │
        No
        ↓
    Revise hypothesis. Do not repeat same technique.
```

## Core Principles

### 1. Manual Therapy Tests Hypotheses
Every manual technique should have a clear rationale:
- **Before:** "I hypothesize that C5-6 hypomobility is contributing to this patient's neck pain."
- **Technique:** C5-6 posterior-anterior mobilization (Grade III-IV)
- **Expected response:** Immediate improvement in cervical rotation and/or pain reduction
- **If no response:** Hypothesis is not supported → revise, do not repeat

### 2. Manual Therapy Is Not the Endpoint
Manual therapy creates a **window of opportunity**:

| Manual therapy provides... | Which enables... |
|---------------------------|-----------------|
| Pain reduction | Exercise at higher intensity/range |
| Increased ROM | Strengthening through full range |
| Reduced muscle guarding | Motor control retraining |
| Neurophysiological effects | Confidence to move |

If the patient only improves during the session but returns to baseline by next visit → the manual therapy is not driving lasting change. Active intervention must fill the gap.

### 3. Response Should Carry Over
| Carryover Pattern | Interpretation | Action |
|-------------------|---------------|--------|
| Immediate relief lasting >24 hours | Manual therapy supports hypothesis + active rehab working | Continue combined approach |
| Immediate relief but returns by next session | Manual therapy alone insufficient | Increase active intervention emphasis |
| No immediate response | Hypothesis not supported | Revise hypothesis, change technique or target |
| Worsening after manual therapy | Irritable condition, wrong technique, or wrong diagnosis | Reassess, reduce intensity, consider alternative |

### 4. Dependency Is a Treatment Failure

**Warning signs of manual therapy dependency:**
- Patient improves during session but always returns to baseline
- Patient requests specific techniques by name ("I need my neck cracked")
- Treatment course extends beyond 8-10 sessions with manual therapy as primary intervention
- Patient cannot function without regular manual therapy sessions
- No progression toward active self-management

**If dependency develops:** This is a decision point. Transition to active intervention. Educate the patient about self-management. The goal is therapist obsolescence, not therapist dependence.

## When Manual Therapy Is Appropriate

| Scenario | Rationale |
|----------|-----------|
| Acute joint stiffness limiting function | Restore ROM to enable exercise |
| Post-surgical adhesion/stiffness | Complement active rehab |
| Acute muscle guarding/spasm | Reduce guarding to allow movement |
| Testing a mechanical hypothesis | Controlled perturbation to test hypothesis |
| Facilitating exercise in pain-sensitive patients | Window of reduced pain for exercise |
| Neurodynamic component | Neural mobilization as part of hypothesis test |

## When Manual Therapy Is NOT Appropriate

| Scenario | Why Not |
|----------|---------|
| Red flags present | Safety: do not manipulate/mobilize |
| Chronic pain with central sensitization as primary driver | Passive treatment reinforces illness behavior; education + graded exercise is primary |
| Patient already mobile with full ROM | Nothing to mobilize — wrong hypothesis |
| Replacing active intervention long-term | Creates dependency |
| Patient expects "fixing" from passive treatment | Misaligned expectations — educate first |
| Hypermobility | Mobilization of already hypermobile joints worsens instability |

## Specific Precautions

### Cervical Spine
- **Canadian C-Spine Rule** before any cervical manual therapy after trauma
- **VBI screening** before upper cervical techniques
- **No high-velocity manipulation** without specific training and informed consent
- **Cervical myelopathy → absolute contraindication** to cervical manipulation

### Lumbar Spine
- **Cauda equina screening** before mobilization
- **Osteoporosis risk** — modify technique, reduce force
- **Post-surgical** — respect surgical precautions and healing timelines

### Peripheral Joints
- **Acute inflammation** — gentle techniques only, respect tissue irritability
- **Joint replacement** — within permitted ROM, no manipulation
- **Fracture risk** — imaging if suspected before any manual intervention

## Dosing Principles

| Parameter | Acute/Irritable | Subacute | Chronic/Stiff |
|-----------|----------------|----------|---------------|
| Grade | I-II (pain-free range) | II-III (into resistance) | III-IV (into resistance/stiffness) |
| Duration | 30-60 seconds per technique | 1-2 minutes per technique | 2+ minutes per technique |
| Repetitions | 2-3 sets | 3-4 sets | 4-6 sets |
| Reassess after | Each set | Each technique | End of session |
| Active exercise after | Gentle AROM | Progressive AROM/PROM | Strengthening into new range |

## Integration with Active Rehabilitation

The ideal treatment session structure:

1. **Assess** — Identify today's primary limitation
2. **Manual therapy** (if indicated) — Target the limitation
3. **Reassess** — Has the limitation changed?
4. **Active exercise** — Use the new ROM/reduced pain to exercise
5. **Home program** — Active exercises the patient does between sessions

**Session progression over course of treatment:**

| Phase | Manual Therapy % | Active Exercise % |
|-------|-----------------|------------------|
| Sessions 1-2 | 50-60% | 40-50% |
| Sessions 3-5 | 30-40% | 60-70% |
| Sessions 6-8 | 10-20% | 80-90% |
| Sessions 8+ | 0-10% (if needed) | 90-100% |

## Decision Criteria for Manual Therapy

### Continue manual therapy if:
- It consistently produces lasting improvement (>24 hour carryover)
- It enables active exercise that the patient cannot do otherwise
- Active intervention is progressing alongside

### Stop manual therapy if:
- No within-session response after 2-3 applications
- Carryover never exceeds 24 hours despite active rehab
- Patient becoming dependent
- Active intervention alone produces equivalent results
- Treatment course exceeding 8-10 sessions with manual therapy still as primary
