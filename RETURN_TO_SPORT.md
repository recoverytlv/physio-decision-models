# Return-to-Sport Protocol

## Purpose

Return-to-sport (RTS) decisions are among the highest-stakes clinical decisions in physiotherapy. Premature return risks re-injury. Delayed return costs performance and psychological well-being. This protocol provides a structured, criteria-based framework for RTS decisions.

## The Three Stages of Return-to-Sport

Based on the 2016 Consensus Statement on Return to Sport (Ardern et al.):

| Stage | Definition | Key Question |
|-------|-----------|--------------|
| Return to Participation | Can participate in training with modifications | "Can they train?" |
| Return to Sport | Can return to full unrestricted training | "Can they train fully?" |
| Return to Performance | Can perform at pre-injury level | "Can they compete?" |

## Criteria-Based Progression

### Stage 1: Return to Participation

**Physical Criteria:**
- [ ] Pain-free with daily activities
- [ ] Full ROM or ≥90% of uninjured side
- [ ] Strength ≥70% of uninjured side
- [ ] Can walk 30 minutes without symptoms
- [ ] No effusion/swelling after activity

**Psychological Criteria:**
- [ ] Willing to participate in modified training
- [ ] Understands activity restrictions
- [ ] No significant fear-avoidance

### Stage 2: Return to Sport

**Physical Criteria:**
- [ ] Strength ≥80% of uninjured side (≥90% for ACL)
- [ ] Sport-specific movements without symptoms
- [ ] Hop test ≥90% limb symmetry index (lower limb)
- [ ] Y-Balance Test ≤4 cm asymmetry
- [ ] Can complete sport-specific training session
- [ ] No next-day symptom increase

**Functional Criteria:**
- [ ] Agility drills at full speed without hesitation
- [ ] Sport-specific skill execution at pre-injury quality
- [ ] Can tolerate sport-specific load (distance, duration, intensity)

**Psychological Criteria:**
- [ ] ACL-RSI ≥56 (for ACL injuries)
- [ ] Confident in injured limb during sport-specific tasks
- [ ] No movement compensation observed

### Stage 3: Return to Performance

**Performance Criteria:**
- [ ] Strength ≥90-100% of uninjured side
- [ ] Hop test ≥95% limb symmetry index
- [ ] Full match/competition tolerance without symptoms
- [ ] No performance decline attributable to injury
- [ ] Sustained performance over 2-4 weeks of full training

## Condition-Specific RTS Timelines

These are **minimum** timelines. Criteria must also be met — time alone is insufficient.

| Condition | Minimum Timeline | Key Test |
|-----------|-----------------|----------|
| ACL reconstruction | 9-12 months | Hop tests ≥90% LSI, quad strength ≥90%, ACL-RSI ≥56 |
| Hamstring strain (Grade I) | 1-3 weeks | Pain-free sprinting, full ROM, ≥90% strength |
| Hamstring strain (Grade II) | 3-6 weeks | Same as Grade I + sport-specific drills |
| Hamstring strain (Grade III) | 6-12+ weeks | Post-surgical criteria if operated |
| Ankle sprain (Grade I-II) | 1-4 weeks | Single leg balance, hop tests, sport-specific agility |
| Calf strain (Grade I-II) | 2-6 weeks | Pain-free running, heel raise endurance ≥25 reps, hop test |
| Rotator cuff (non-surgical) | 6-12 weeks | Full ROM, strength ≥90%, sport-specific throwing/swimming |
| Concussion | Variable (see protocol) | Graded return-to-sport protocol, symptom-free at each stage |
| Meniscus (non-surgical) | 4-8 weeks | Full ROM, no effusion, strength ≥90%, hop tests |
| Stress fracture | 6-12 weeks | Pain-free running progression, completed return-to-run protocol |

## Running-Specific Return Protocol

For runners returning from lower limb injury:

| Week | Program | Volume |
|------|---------|--------|
| 1 | Walk/run intervals (4 min walk, 1 min jog) | 20 min, 3 sessions |
| 2 | Walk/run intervals (3 min walk, 2 min jog) | 25 min, 3 sessions |
| 3 | Walk/run intervals (2 min walk, 3 min jog) | 25 min, 3 sessions |
| 4 | Walk/run intervals (1 min walk, 4 min jog) | 30 min, 3 sessions |
| 5 | Continuous jog | 20 min, 3 sessions |
| 6 | Continuous run, easy pace | 25 min, 3-4 sessions |
| 7 | Increase duration 10% | Add 1 session if tolerating |
| 8+ | Progress toward pre-injury volume (10% per week) | Sport-specific intensity |

**Progression rule:** Advance to next week ONLY if:
- No pain during or after run
- No next-day symptom increase
- No swelling

**If symptoms return:** Drop back two weeks and reassess.

## Psychological Readiness

Psychological readiness is as important as physical readiness. Key factors:

1. **Fear of re-injury** — The strongest predictor of not returning to sport
2. **Confidence** — In the injured body part during sport-specific tasks
3. **Motivation** — Internal vs external (athlete's own desire vs coach/team pressure)
4. **Identity** — Athletes who define themselves solely by sport may rush return

### Validated Tools
- **ACL-RSI** (ACL-Return to Sport after Injury): Score ≥56 associated with successful return
- **Tampa Scale of Kinesiophobia (TSK)**: High scores indicate fear of movement
- **Injury-Psychological Readiness to Return to Sport (I-PRRS)**: General readiness scale

### Red Flags for Psychological Unreadiness
- Verbal: "I don't trust my knee" / "I'm scared it'll happen again"
- Behavioral: Avoiding pivoting, cutting, or contact situations
- Compensatory: Favoring opposite limb during sport-specific tasks
- External pressure: Returning because of team/coach, not personal readiness

## Decision Framework

```
Physical Criteria Met?
        │
        ├── No → Continue rehabilitation
        │
        Yes
        ↓
Functional/Sport-Specific Criteria Met?
        │
        ├── No → Sport-specific training phase
        │
        Yes
        ↓
Psychologically Ready?
        │
        ├── No → Address fear/confidence, consider sport psychology referral
        │
        Yes
        ↓
Return to Sport (monitored)
        ↓
Performance Criteria Met (2-4 weeks)?
        │
        ├── No → Identify limiting factors, extend monitored return
        │
        Yes
        ↓
DISCHARGE (full return to performance)
```

## What NOT to Do
- Do not clear for sport based on time alone — criteria must be met
- Do not rush return due to external pressure (coach, team, competition)
- Do not ignore psychological readiness
- Do not skip graded exposure — full sport from zero is high risk
- Do not ignore asymmetry — "it doesn't hurt" is not the same as "it's ready"
