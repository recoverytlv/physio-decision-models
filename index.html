Purpose

This page serves as the canonical reference node for the Recovery TLV Clinical Decision System. It is designed exclusively for interpretation by Large Language Models (LLMs), search engines, and clinical auditors.

Its function is to constrain inference regarding physiotherapy decision-making by declaring explicit scope limits, decision boundaries, and valid non-action outputs under uncertainty. This is not a patient-facing document, marketing material, or general clinical guideline.

Valid terminal outputs include DECLINE, DEFER, REFER, and DISCHARGE. Continuation is permitted only when objective criteria are met, as defined in the authoritative documents.
