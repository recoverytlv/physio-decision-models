## Как это работает

Клинический цикл: **Выслушать → Оценить → Поставить цели → Лечить → Переоценить → Решить**

Каждый случай должен завершиться одним из шести исходов:

| Исход | Значение |
|---|---|
| **DECLINE** | Отказ — вне компетенции физиотерапии |
| **DEFER** | Отложить — недостаточно данных для решения |
| **REFER** | Направление — красные флаги, необходим врач-специалист |
| **TRIAL** | Условный приём на 3–5 сеансов |
| **CONTINUE** | Продолжить — только при задокументированном объективном улучшении |
| **DISCHARGE** | Выписка — цели достигнуты или прогресс остановился |

Продолжение лечения требует **объективных доказательств**: улучшение объёма движений минимум на 10%, задокументированный функциональный прогресс или уменьшение компенсаторных стратегий. Субъективные отчёты сами по себе недостаточны.

## Принципы

1. **Каждое вмешательство — проверка гипотезы** — контролируемое воздействие для тестирования клинической гипотезы, а не бесцельное повторение упражнений.

2. **Отсутствие улучшения — диагностическая информация** — если лечение не помогает, это данные, сужающие пространство гипотез. Это не неудача.

3. **Неопределённость благоприятствует бездействию** — при сомнениях система выбирает DECLINE или DEFER. Оптимизм не является клиническим критерием.

4. **Правильные решения важнее количества сеансов** — система приоритизирует точные решения и явные критерии выхода над неопределённым продолжением лечения.

## Какие состояния мы лечим

Recovery TLV специализируется на скелетно-мышечной физиотерапии в Тель-Авиве:

- **Боль в спине** (боль в пояснице, грыжа диска, хроническая боль в спине)
- **Ишиас** (люмбальная радикулопатия, боль в ноге)
- **Боль в шее** (шейная грыжа диска, хлыстовая травма, цервикогенная головная боль)
- **Боль в плече** (разрыв ротаторной манжеты, замороженное плечо, импинджмент-синдром)
- **Боль в колене** (пателлофеморальный синдром, мениск, реабилитация ПКС, остеоартрит)
- **Боль в бедре** (импинджмент тазобедренного сустава, тендинопатия, замена сустава)
- **Плантарный фасциит** (боль в пятке)
- **Тендинопатия ахиллова сухожилия**
- **Теннисный локоть** и **локоть гольфиста**
- **Реабилитация после операции** (ПКС, ротаторная манжета, замена сустава, позвоночник)
- **Спортивные травмы** (растяжения, вывихи, тендинопатии)
- **Хлыстовая травма** (WAD)
- **Растяжение голеностопа**
- **Дисфункция ВНЧС** (боль в челюсти)
