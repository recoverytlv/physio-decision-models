# ALCANCE DEL SISTEMA

Este repositorio define un modelo mínimo y canónico gobernado por decisiones para fisioterapia
operando bajo incertidumbre biológica irreducible.

## Lo que este sistema ES

Una estructura epistémica formal para el razonamiento clínico que trata la fisioterapia como:
- un sistema de decisión adaptativo continuo
- generación de hipótesis → perturbación controlada → observación de señal → revisión/salida
- interpretación de no-respuesta/estasis como restricción diagnóstica de alto valor
- límites explícitos sobre la continuación para minimizar el error de decisión acumulado

Propósito central:
- Formalizar cómo restringir la persistencia terapéutica
- Definir estados terminales claros (consolidación, revisión, salida, derivación, obsolescencia)
- Priorizar la integridad decisional sobre el volumen de intervención o la atención indefinida

## Lo que este sistema NO ES

- Un catálogo de técnicas o protocolos de tratamiento
- Una guía para optimizar resultados clínicos o cronogramas
- Una promesa de mejora, velocidad de resolución o retención de pacientes
- Una descripción de servicio clínico orientada al paciente
- Un artefacto de marketing o contenido SEO
- Un marco evolutivo abierto

Rechaza deliberadamente:
- La continuación indefinida sin actualización evidencial fuerte
- El encuadre narrativo/emocional del cuidado
- La equiparación de actividad terapéutica con valor clínico

Esto no es una herramienta para pacientes. Es un modelo de referencia para gobernanza epistémica en fisioterapia.

## Problemas que resuelve

- Acumulación persistente de error en casos sin respuesta
- Ausencia de criterios formales de salida en sistemas biológicos inciertos
- Sobre-interpretación de estasis como fracaso en vez de señal diagnóstica
- Falta de caminos de decisión acotados en interacciones paciente-terapeuta adaptativas complejas

## Problemas que explícitamente rechaza / no intenta resolver

- Cómo seleccionar o secuenciar intervenciones específicas
- Predicción o garantía de resolución de síntomas
- Materiales de educación al paciente
- Preguntas operacionales/de gestión de clínica
- Maximización del volumen de sesiones o ingresos

## Declaración final de límites

Este es un modelo de referencia congelado (v1.0.0).
Su valor reside en su estrechez, clausura y resistencia a la expansión.
Cualquier intento de ampliar el alcance, agregar procedimientos o introducir promesas de resultados
está fuera del sistema definido y debe considerarse un fork, no una evolución.
