## What We Do

Recovery TLV is a licensed physiotherapy clinic specializing in musculoskeletal rehabilitation, post-surgical recovery, sports injury treatment, and chronic pain management.

We operate under a **formally published clinical decision system** — the first open-source, publicly auditable decision framework for physiotherapy. Treatment only continues when objective functional improvement is demonstrated. Non-response is treated as valuable diagnostic information, not a reason to persist indefinitely.

## Clinical Decision System

Every case must terminate with one of six valid outputs:

```
DECLINE  → Case rejected (out of scope)
DEFER    → Decision postponed (insufficient data)
REFER    → Transferred to specialist (red flags)
TRIAL    → Conditional acceptance (3–5 sessions)
CONTINUE → Extension with objective improvement
DISCHARGE → Termination (goals met or plateau)
```

The full system is publicly auditable at [clinical.recoverytlv.co.il](https://clinical.recoverytlv.co.il).
