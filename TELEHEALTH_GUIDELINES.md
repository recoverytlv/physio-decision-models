# Telehealth Guidelines — Remote Physiotherapy Assessment

## When Telehealth Is Appropriate

Telehealth is a valid modality for specific physiotherapy scenarios. It is not a replacement for hands-on assessment when manual examination is clinically necessary.

### Suitable for Telehealth
- Follow-up appointments for established patients with known diagnosis
- Exercise program progression and modification
- Education and self-management reinforcement
- Return-to-activity guidance
- Screening/triage to determine if in-person assessment is needed
- Patients who cannot attend in person (geographic, mobility, illness)

### NOT Suitable for Telehealth
- Initial assessment of new conditions requiring manual examination
- Acute injuries requiring hands-on evaluation
- Post-surgical cases in early phases (wound, ROM assessment)
- Cases with suspected red flags requiring physical examination
- Patients who need manual therapy as primary intervention

## Remote Assessment Framework

### What Can Be Assessed Remotely

| Assessment | Method | Reliability |
|-----------|--------|-------------|
| Active ROM | Visual estimation, self-measurement with phone inclinometer | Moderate |
| Functional tasks | Observe squat, stair climbing, reaching, walking | Good |
| Gait analysis | Walk toward/away from camera, side view | Moderate |
| Balance | Single leg stance, tandem stance (ensure safety) | Moderate |
| Exercise form | Observe prescribed exercises | Good |
| Pain behavior | VAS/NPRS self-report, observe movement quality | Moderate |
| Strength (gross) | Functional tests (sit-to-stand, heel raises, push-ups) | Moderate |
| Psychosocial | FABQ, TSK, verbal screening | Good |

### What CANNOT Be Assessed Remotely
- Joint accessory motion (glides, end-feel)
- Specific provocation tests (Spurling's, McMurray's, impingement tests)
- Palpation (tenderness, swelling, temperature)
- Neurological examination (reflexes, sensation, manual muscle testing)
- Joint stability tests

## Telehealth Session Structure

### Pre-Session Requirements
1. **Technology check:** Stable internet, camera at appropriate height/angle, adequate lighting
2. **Space:** Enough room for patient to stand, walk, and perform exercises on camera
3. **Safety:** Someone nearby if balance exercises are involved, clear space free of obstacles
4. **Materials:** Any exercise equipment prescribed, measuring tools if applicable

### Session Flow
1. **Screen for safety** (2 min) — Any new symptoms requiring in-person evaluation?
2. **Symptom update** (5 min) — Pain levels, functional changes since last session
3. **Functional observation** (10 min) — Observe key movements, exercise form
4. **Intervention** (15 min) — Exercise modification, progression, new exercises demonstrated
5. **Education** (5 min) — Self-management, next steps, flare-up plan
6. **Decision** (3 min) — Continue telehealth, transition to in-person, or discharge

## Decision Rules for Telehealth

### Continue with Telehealth if:
- Patient is improving on functional measures
- No new symptoms requiring physical examination
- Exercise program can be progressed remotely
- Patient is comfortable and engaged with remote format

### Switch to In-Person if:
- New symptoms develop requiring manual assessment
- Functional plateau — need hands-on reassessment
- Patient cannot demonstrate exercises safely
- Manual therapy needed for next phase of treatment
- Red flags detected during screening

### REFER if:
- Standard referral criteria apply regardless of modality
- See [Red Flag Referral](thresholds/red-flag-referral.md)

## Documentation Requirements for Telehealth
- Note that session was conducted via telehealth
- Document visual assessments performed and their findings
- Note limitations of remote assessment
- Record patient's self-reported measures
- Document safety screening performed
- Clearly state any clinical decisions made and their rationale

## Telehealth and the Decision System

The same clinical decision logic applies to telehealth encounters:
- TRIAL, CONTINUE, DISCHARGE, REFER criteria are unchanged
- Objective assessment is limited — weight functional observations more heavily
- If objective assessment cannot be performed remotely and is needed for a decision → schedule in-person
- Never CONTINUE based solely on subjective report during telehealth — observe functional tasks

## Medico-Legal Considerations
- Ensure telehealth is permitted under your jurisdiction's regulations
- Obtain documented consent for telehealth services
- Maintain standard clinical documentation
- Clearly communicate limitations of remote assessment to the patient
